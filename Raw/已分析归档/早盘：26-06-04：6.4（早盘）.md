# 6.4（早盘）

> B站充电专栏 | 2026-06-04

一、指数纪律（最高优先级）
日线顶部结构已走到第16个交易日，今天严格执行机械化标准，不预判、不抢跑：

收盘价跌穿 4033 点 → 清仓

向上突破 4130 点 → 满仓

两点之间：维持现有仓位，不动作

心理承受力弱的，今天可以不看盘，只设两个预警价位即可，盯住 4033 这条生命线。

二、量能是核心矛盾
昨天创业板指冲高近4%尾盘只收涨1.65%，跳水的根子在量能。午盘前后预估成交额始终横在 2.9–3万亿 之间，指数走那么强都没能等到增量资金进场，提前获利了结就不奇怪了。

3万亿这个体量本身很尴尬，像个巨型"火药桶"——砸下来很难接稳，继续边涨边放量收盘又奔3.5万亿。所以现阶段无论涨跌，开盘缩量都算"安全牌"。今天要止住颓势，关键看两点：量能能否上台阶，以及"易中天"这个基本盘稳不稳（昨天这三家全天贡献了创业板指67个点里的50个点，尾盘要不是它们撑着差点翻绿）。

另一个要留心的信号：煤炭、电力这类防御板块如果仍像昨天那样蠢蠢欲动，可能是资金转守的苗头。

三、市场结构观察
连续两天"一九行情"，指数大涨但个股跌多涨少，午后没回落时涨停也才六七十家。

放量了，但涨停个股中成交额超50亿的只有4只，比周二还少1只——短线大资金在收缩、趋利避害。

好消息：跌停家数终于降到10家以内。这是此前定的"赚钱效应和中小盘风格企稳"的积极信号，今天看这个指标能否稳住，是中小盘能否延续的关键。

四、板块策略
AI方向：两天反弹后，多数个股逼近前高或连涨多日后拉出较长上影线。今天预期弱修复，未来几天大概率震荡为主。不追高，等待确认。

半导体方向：经历6个交易日分歧后，昨天封测带头反弹，预期反弹还能延续，今天重点观察强度。这是当前相对有持续性的方向。

五、题材落地
光通信 / 光互联（当前阵眼） 今天关键看资金对光互联的反馈。昨天开盘主攻的铜缆、光纤方向很精准，冷门票裕太微（阵眼）资金认可度高。

催化：英伟达6月2日官宣 Spectrum-X Ethernet Photonics 全面量产，基于硅光+CPO架构，支撑 Vera Rubin AI Factory 扩展；Computex 上黄仁勋与迈威尔CEO同台，把AI瓶颈指向"高速连接"，光互连进入景气周期。

风险：最大对手来自内部分流。"易中天"、联讯仪器、长进光子等高价大票已近乎独立成板，资金一旦漂移到大票，盘面就容易失控。MLCC（风华高科、火炬电子最多二连板）、AIPC被机构吹高但持续性一般，注意板块内部跷跷板。

AI Agent / CPU 重构 Computex 行程过半，黄仁勋"AI Agent时代已全面到来"、高通安蒙"2026是Agent之年"，英特尔与Arm齐声强调 Agent 对 CPU 的催化（CEO打电话催货、需求升温超预期）。

逻辑：推理场景 CPU 耗时占比 50%–90%，TrendForce 测算每GW所需CPU核心数从3000万激增至1.2亿；2026年服务器CPU量价齐升、高双位数增长，消费级随AIPC温和增长但弹性弱。

相关：兴森科技（IC封装基板）、禾盛新材（参股熠知电子，国产CPU）。

人形机器人 比亚迪入局：李柯确认在开发人形机器人，"尧舜禹"项目已迭代到第七代，截至5月约150台上岗工厂，目标2026年内部自用2万台，未来或走经销商渠道、做开放平台。国金证券陈传红：2026是人形机器人0-1兑现关键节点，国产头部出货有望从数千台跨到数万台。

相关：新时达（四大具身产品矩阵）、华培动力（六维力传感器，已获比亚迪定点）。

AI软件 / Agent生态 Meta涨4.24%，推出"Meta Business Agent"，企业可用AI代理回复客户消息，Agent生态繁荣有望全面升级软件行业。A股相关：美格智能、易点天下。

防御板块（电力/煤炭） 炒作已进入挖掘补涨阶段：金房能源前一天连板后无人问津，绝对高度从大唐发电6板降到华电能源5板、粤电力A4板。若机构股趋于理性，这些防御板块可能逐渐失去生态位——资金最近已在低价股、ST股尝到甜头（节能铁汉、ST闻泰、ST联翔）。

今日一句话：指数盯死4033，方向看量能能否破3万亿+"易中天"稳不稳；题材主攻光互联看强度，半导体看封测反弹延续性，AI弱修复不追高。
