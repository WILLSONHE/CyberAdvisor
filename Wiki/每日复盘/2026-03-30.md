# 2026-03-30 如何看年报：F10实战

> 以金风科技为例 | 全产业链联动 | 绿色甲醇新业务测算

## 一、年报阅读四步法

| 步骤 | 内容 |
|------|------|
| **管理层** | 有无董事无故缺席？关键岗位缺席可能爆雷 |
| **主营业务** | 营收、净利、行业地位。龙头业绩大幅增长→全行业可能迎来转机 |
| **F10财务** | 毛利/净利同行对比、偿债能力、运营能力、杜邦ROE |
| **产品结构** | 向大型化高端化升级→毛利净利改善 |

## 二、全产业链联动

**龙头业绩好转 → 全行业变化。** 研究产业的逻辑：先找行业龙头 → 沿龙头产品线向上游挖原材料 → 全产业链贯通。

以风电为例：整机（金风科技 730亿）→ 叶片（中材科技，业绩比金风更亮眼）→ 塔筒（大金重工，趋势大牛）。

## 三、绿色甲醇新业务深度

### 背景
- 在手订单 75 万吨，与马士基签订年产 50 万吨长期采购协议
- 中远海运预测：2030 年航运业绿色甲醇需求达 **1 亿吨**

### 金风科技成本优势
- 风电 24 小时连续供电 vs 光伏仅白天→风电耦合制成本更低
- 风电制甲醇成本 ~2200 元/吨（vs 光伏制 ~3800 元/吨）
- 选址兴安盟：一类风能资源区 + 东北玉米秸秆充裕 + 靠近港口

### 利润测算
| 参数 | 数值 |
|------|------|
| 售价 | ~6000元/吨 |
| 单吨利润 | ~2600元 |
| 一期25万吨年利润 | ~6.5亿 |
| 二期50万吨达产 | ~13亿 |

全球最大生物质耦合绿氢项目 = 兴安盟（唯一），马士基摩洛哥大项目 2028-2029 才投产。**金风科技可吃三年独占期。**

## 四、投资哲学

看年报的目的：找到业绩好、有增量、不受突发事件影响的龙头公司。如果四月杀跌把金风从 1000 亿干到 600-700 亿：1000→2000 是 1 倍，600→2000 是 3-4 倍。
