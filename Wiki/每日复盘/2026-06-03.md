## 一、当日定调

> 「盘面内生结构是向上的，下午的下跌是外力（关税消息+量化踩踏）打破了内部结构平衡，而非主线瓦解导致的内生性调整。」

| 维度 | 判断 |
|------|------|
| 盘面 | 早盘高歌猛进 → 下午急转直下 → 尾盘收回不少 |
| 性质 | **外部扰动**，非趋势反转 |
| 后市 | 消化情绪后大概率仍按内生性逻辑向上，路径更曲折 |

### 回调两层原因

| 层级 | 原因 |
|------|------|
| **内生** | 轮动市中单一方向涨太快，上午过热需健康回调 |
| **外力** | 彭博 12.5% 关税传闻（上午已有）+ 13:37 欧盟对华贸易战消息 → 群扩散放大情绪 → 量化策略踩踏 |

> 「按上午正常节奏，主升结构已成型，正常回落不太可能出现调整级下杀——只有外力才能打破内部结构平衡。」

**关税评估**：12.5% 力度弱于此前且能否通过存疑；光连接（光纤/CPO）走全球产业链共振，一条关税传闻不改产业趋势。

## 五、交易方法论：亨通光电接力

> 「选接力标的时，板块内部要有支撑——要么板块指数往上，要么板块中军（长飞光纤）稳定。」

| 节点 | 细节 |
|------|------|
| 盘前 | 锁定光纤阵型 → 亨通；MLCC（风华高科）午后两次开板失去竞争力 |
| 竞价 | 亨通高开 **8.3%**，中军长飞光纤同步高开，阵型成立 |
| 开盘 2 分钟 | 亨通有抛压，长飞持续往上 → 板块正向循环 |
| 结果 | 承接抛压后迅速封板，接力胜率高 |

**提炼**：中军稳 → 龙头高开抛压快速消化 → 封板。
