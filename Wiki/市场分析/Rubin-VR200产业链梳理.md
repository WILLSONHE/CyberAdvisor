# Rubin VR200 产业链梳理（2026-05-22）

> 大摩 5/20 研报拆解：ODM 口径整机 **780 万美元**（GB300 +95%）| GPU 占比降至 51% | 内存 +435%、PCB +233%、MLCC +182%

---

## 一、整机价值量跃迁

| 代际 | ODM 整机价 | GPU 占比 | 关键变化 |
|------|-----------|---------|---------|
| GB200 | 333 万美元 | ~65% | 传统服务器逻辑 |
| GB300 | 399→700 万美元 | ~63% | AI 机柜高价值密度 |
| **VR200 (Rubin)** | **780 万美元** | **~51%** | 全组件通胀，非单点升级 |

OEM（联想/华硕/技嘉/戴尔）采购价更高——含品牌、服务溢价。

---

## 二、组件涨价矩阵（VR200 vs GB300）

| 组件 | GB300 | VR200 | 涨幅 |
|------|-------|-------|------|
| GPU | 252 万 | 396 万 | +57% |
| **内存** | 37 万 | **200 万** | **+435%** |
| NVLink Switch | 6.48 万 | 14.4 万 | +122% |
| 网络芯片 | 26.1 万 | 57.6 万 | +121% |
| **PCB** | 3.5 万 | **11.6 万** | **+233%** |
| **MLCC** | 1530 | 4320 | +182% |
| ABF 载板 | 1.1 万 | 2 万 | +82% |
| 电源 | 5.7 万 | 7.6 万 | +32% |
| 散热/液冷 | 6.4 万 | 7.2 万 | +12% |

**PCB 涨价逻辑**：Compute Board 22→26 层 HDI、CCL M7→M8；Switch Tray 24→32 层；新增 Midplane（18 块×1500 美元）、ConnectX PCB、BlueField PCB。

**MLCC 逻辑**：107 规格高容 MLCC 毛利可达 70%，国内尚未完全自主量产 → 通胀型材料 + 涨价周期。

**电源/液冷**：机柜功率 120–140kW → 200kW+（Rubin Ultra 至 600kW）；架构向 **400V AC → 800V DC HVDC** 演进。VR200 **全液冷无风扇**——液冷从选配变标配。

**ODM**：组装测试价值 GB300 10.82 万 → VR200 14.96 万（+38%）；2026Q2 或转向**寄售模式**减轻 ODM 库存压力。

## 五、MicroLED 光互联（新方向）

京东方 A × 康宁（5/20 MOU）：玻璃载板、光互联应用。

**范式**：数百并行低速通道替代少数高速通道（宽而慢）→ 低功耗（可达铜缆 5%）、高可靠性。

| 公司 | 角色 |
|------|------|
| 华灿光电 | 京东方旗下 MicroLED 核心；6 英寸量产线；光通信 4 家客户进展 |
| 兆驰股份 | LED 垂直一体化 → 光通信芯片+模块全链 |
| 思特威 | TX/RX MicroLED + CMOS 四类芯片协同 |
| 聚光科技 | 晶圆级微纳光学；收购 SMO 强化微透镜阵列 |
| 茂莱光学 | **双曲线**：半导体光刻光学 + AI 光互联；半导体营收占比 58%（2025） |
