> 内容源技术教学系列：**单根 K 线 → 组合 K 线 → 形态 → 趋势**（约 4–6 个月，每周一节）。

## K 线基础

- **阳线**：收盘 > 开盘；**阴线**相反
- **下影线** = 最低价；**上影线** = 最高价
- **光头光脚**：开盘≈极值、收盘≈另一极值，实体大，影线极短（不必几何完美）

### 涨跌幅参照（指数 vs 个股）

| 指数长红/长黑 | 含义 | 个股约等于 |
|---------------|------|------------|
| >1% | 明显 | 3–5% |
| >3% | 很强 | 接近涨停 |
| >5% | 历史级 | 批量涨停/跌停 |

## 长红线（光头光脚阳线）— 四种情境

### 1. 低档长红（多方启动）

**前置**（四条）：

1. 真实下跌：一般 **5–10 交易日以上**，越长越好
2. 跌幅够深：个股约 **15%**，指数约 **5%**
3. 突发标准长红，实体显著
4. **放量**为强信号；恐慌底可缩量仍有效

**叙事**：卖压多日耗尽 → 大资金一日吃尽卖盘拉至收盘最高 → **短期底部信号**（涨多久不猜）。

**案例**：茅台 2026-02 长期阴跌后长红（买） vs 已涨一段后的长红（卖）。

### 2. 高档长红（多方耗尽警告）

**特征**：长期温和小阳后，突发**远超以往**的光头光脚大阳，可带**跳空**。

**逻辑**：最后一批观望/犹豫资金入场 → 买盘当日耗尽 → 多为**顶**而非新升浪。

**口诀**：「不见长虹不回头」——跳空光头阳常为顶。

**操作**：不追；可分批减仓 + 止损。茅台 2400 后再未新高；中芯国际上涨后出现类似形态 → **越涨越卖**，非一次性清仓（可能还有最后一波）。

### 3. 箱体向上突破

- 箱体内多空拉锯数周/月 → **放量**长红突破上沿 = 偏多，盘整结束
- **必须放量**；缩量突破易假突破跌回箱内
- **目标参考**：箱体高度翻一倍（小箱/大箱分别估算，作计划非精确点位）
- 到目标位可分批出；突破后收难看阴线应离场

### 4. 长红被跌破（风控纪律）

长红 = 当日买入者成本区 = 支撑。

| 防线 | 动作 |
|------|------|
| 实体**中点** | 警戒，**先减仓** |
| 实体**最低点** | **无条件清仓** |

无纪律的技术分析 = 赌博。例：长红次日跌破 → 直接走，避免后续阴跌。

## 长黑线（光头光脚阴线）— 三种情境

> 长黑只有三种（无「长黑被跌破」对称项——长黑本身已是危险信号）。

### 1. 高档长黑（无条件清仓）

上涨末端突发光头光脚长黑：大资金获利了结 / 利空 / 技术恶化 / 大盘崩塌——**原因不重要，价格包含一切**。

**纪律**：**无条件清仓**，不赌反弹。5/14 上证、个股跌停式长黑、标普 500 同类。**客观事实 > 主观判断**。

### 2. 低档长黑（底部反弹信号，非反转）

下跌末端 + **非理性加速**：连续大阴、跌幅远超平时、恐慌极致（924/中际旭创 180→60 类）。

**叙事**：最后一批散户/弱机构砍仓 → 卖盘极致 → 往往接近短期底。

**注意**：

- 是**反弹**信号，**不是**趋势反转保证
- 只能**小仓**博弈 + **严格止损**（破长黑最低点出；反弹至跌幅一半可止盈）
- 不可满仓 all-in 当反转做

**对照**：低档长红 = 多方主动反攻；低档长黑 = 空方力量枯竭——同一低位，两种叙事。

### 3. 箱体向下突破（翻箱向下）

长黑跌破箱体下沿 → 箱内多头被套 → 止损抛压 → 按箱体高度向下翻目标（熊市案例：金发科技 23.19→18.47，下一目标约 13.75）。

- 向下突破**不必须放量**；缩量下跌有时更危险（无人接盘）
- 放量跌破 = 看空一致 → 哪怕亏损也要离场
- 当前牛市多见向上翻；向下翻等熊市再练

## 影线：上吊 / 铁锤 / 流星 / 倒锤

影线 = 进攻**失败**的记录；失败有时比成功进攻信号更强（揭示对手真实力量）。

### 上吊线（上涨末端，长下影）

- 实体小（阴阳皆可）；下影线约为实体 **2–3 倍**；上影极短
- **叙事**：空方首次把价打到日内深低，收盘被拉回——表面安全，实为**多方控盘松动**
- **次日确认**：需高开高走/长红化解；低开低走 → 减仓 30–50%，不行则清仓
- 当天可不走，**必须看次日**

### 铁锤线（下跌末端）

- 形态与上吊相同，**位置在底部** → 偏多（空方进攻失败）

### 流星线（上涨末端，长上影）

- 实体小；上影线约为实体 2–3 倍；下影极短或无
- **叙事**：多方冲高遇强卖压压回——多方进攻**最后尝试**结束
- 与上吊相反：上吊 = 空方进攻开始；流星 = 多方进攻衰竭
- **组合**：上吊 + 流星 → 「必死无疑」
- **化解更难**：次日需高开高走且**站上流星线最高点**；可先减 **50%**（比上吊 30% 更多）

### 倒锤（下跌末端）

- 形态同流星，**位置在底部** → 多方**首次反击**能力，空方权威松动

## 实战自检（周末作业模板）

| 持仓类型 | 自查 |
|----------|------|
| 科技/硬科技/半导体 | 是否出现上吊、流星、高档长黑 |
| 老登/防御 | 是否出现倒锤、铁锤等底部信号 |

顺势优于逆势；顶部信号化解需超预期利好，当前环境下难度高。
