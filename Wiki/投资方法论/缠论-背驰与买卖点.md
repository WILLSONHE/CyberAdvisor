# 缠论 · 背驰与买卖点

> **背驰系统 + 三类买卖点**实操细目。  
> 上级：[[缠论]] | 结构前提：[[缠论-结构与递归]]

---

## 1. 力度比较（通用）

**背驰本质**：价格极值刷新，**驱动力量**未刷新。

| 维度 | 用法 |
|------|------|
| **MACD 柱面积** | 同向两段，后段红/绿柱面积和 < 前段 |
| **黄白线 DIF/DEA** | 价格新高，|DIF| 峰值更低 |
| **成交量** | 辅助；缩量新高警惕（需结构配合） |
| **斜率** | 后段上涨/下跌更「平」 |

**前提**：比较的两段必须是**同向、可比**（同一中枢的进入/离开，或趋势 c 段 vs 前段）。

---

## 2. 趋势背驰（一买/一卖核心）

### 2.1 结构位置

```
下跌趋势：… → [中枢 N] → 离开向下(c 段) → 一买？
上涨趋势：… → [中枢 N] → 离开向上(c 段) → 一卖？
```

| 条件 | 多（一买） |
|------|------------|
| 1 | 至少 **2 个下跌中枢**（或等价趋势结构） |
| 2 | 最后离开段 **破前低** |
| 3 | **MACD 面积**（或黄白线）**不创新低** |
| 4 | 可选：次级别在极值区出现**盘整背驰** |

### 2.2 一买失败

| 现象 | 应对 |
|------|------|
| 新低无背驰 | 仍属下跌延续 → **不补仓** |
| 小转大 | 小级背驰后大级再杀 → **止损** |
| 中枢下移 | 第二个中枢低于第一个 → 仍空思维 |

---

## 3. 盘整背驰

| 场景 | 比较 |
|------|------|
| 中枢震荡 | **离开段** vs **进入段**（向同一方向） |
| 用途 | 盘整内 T、二买辅助、**非**趋势一买 |

**弱于趋势背驰**：仅保证回到中枢内或触发反向笔，**不保证**反转趋势。

---

## 4. 三类买卖点（IF-THEN 表）

### 4.1 一买（多）

| IF | THEN |
|----|------|
| 下跌趋势 + 最后中枢 + c 段趋势背驰 | 分批试多；止损 = 背驰低点下 |
| 上级仍强空 | 仅**反弹**思维，仓位 ≤ [[仓位管理]] 试探档 |
| T+1 | 当日买，次日才能卖 → 控制仓位 |

### 4.2 二买（多）— **主力形态**

| IF | THEN |
|----|------|
| 一买后反弹形成**向上笔/线段** | 确认下跌段结束 |
| 回调**不破一买低** | 二买成立 |
| 回调进**最后中枢**但不破 ZD | 仍属二买变体 |
| 次级别出现**一买或盘整背驰** | 提高精度 |

**止损**：破一买低或破 ZD（视结构）。

### 4.3 三买（多）

| IF | THEN |
|----|------|
| 中枢 **向上离开**（d 段） | 趋势/盘整突破 |
| 回抽 **低点 > ZG** | 三买确认 |
| 上级趋势多 | 可加仓；否则易假突破 |

**失败**：回抽跌回中枢 → 按**盘整**处理，止损 ZG 下。

### 4.4 一卖 / 二卖 / 三卖

| 类 | 对称条件 |
|----|----------|
| 一卖 | 上涨趋势末 c 段**顶背驰** |
| 二卖 | 一卖后反弹**不过一卖高** |
| 三卖 | 向下离开中枢，回抽**高点 < ZD** |

## 5. MACD 操作细则

| 步骤 | 动作 |
|------|------|
| 1 | 定操作级 MACD（12,26,9 默认） |
| 2 | 标记中枢 a、c 段起止 K |
| 3 | 数 **0 轴下**绿柱面积（一买）或 **0 轴上**红柱（一卖） |
| 4 | 面积缩小 + 结构位 → 背驰 |
| 5 | 次级别 0 轴附近再次确认 |

| 陷阱 | 说明 |
|------|------|
| 0 轴下金叉 | 仅反弹，非一买 |
| 柱缩短但价未新低 | 盘整背驰，非趋势一买 |
| 参数改小 | 假背驰增多 → 勿随意改参 |

## 6. 区间套（精确定位）

**场景**：大级别（如日线）进入**可能的最后背驰段**。

```
日线下跌 c 段
  └─ 60min：进入中枢 → 离开 → 盘整背驰 → 60min 一买
       └─ 15min：同样递归 → 精确入场
```

| 原则 | 内容 |
|------|------|
| 方向 | **大级定多空**，小级只找点 |
| 停止 | 小级买点 + 大级结构完成 **同时** 才满仓位 |
| 失败 | 小级买点但大级无背驰 → 按**短差**处理 |

## 7. 仓位与 [[风控逻辑]] 结合

| 买点 | 建议仓位（相对单票） |
|------|---------------------|
| 一买 | 10%~20% 试探；确认二买再加 |
| 二买 | 30%~50% 主仓区 |
| 三买 | 20%~40% 趋势追加；须上级多 |

| 纪律 | |
|------|--|
| 结构破坏 | 破 ZD/一买低/三买 ZG → **机械减** |
| 指数 [[指数纪律框架]] L3+ | 禁一买抄底，仅二买或空仓 |
| 90min 钝化 + 顶背驰 | 优先减仓，非新开 |
