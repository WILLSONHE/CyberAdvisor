# 缠论 · 结构与递归

> 缠论**结构系统**细目：包含 → 分型 → 笔 → 线段 → 中枢 → 走势递归。  
> 上级：[[缠论]] | 配套：[[缠论-背驰与买卖点]]

---

## 1. 包含关系（算法要点）

**输入**：原始 K 线序列（建议用**不复权**或**前复权**一致处理）。

**步骤**：

1. 若 K1、K2 无包含 → 确定临时方向（K2 高 > K1 高为向上）。
2. 若 K2 包含 K1（或反向）→ 按方向合并为一根虚拟 K：
   - **向上**：高 = max(高1,高2)，低 = max(低1,低2)
   - **向下**：高 = min(高1,高2)，低 = min(低1,低2)
3. 合并后的 K 与下一根继续判断包含，直至序列无包含。

| 易错 | 处理 |
|------|------|
| 方向频繁切换 | 以**前一对非包含 K** 定方向 |
| 一字板 | 仍参与包含；涨跌停日力度比较另议 |
| 除权缺口 | 复权不一致会导致假分型 → 统一数据源 |

## 2. 分型（严格与过滤）

### 2.1 标准顶/底分型

```
顶：H2 > H1 且 H2 > H3，且 L2 通常高于 L1/L3（非必须但增强）
底：L2 < L1 且 L2 < L3
```

### 2.2 过滤规则

| 情况 | 处理 |
|------|------|
| **共用 K 线** | 顶与底共用中间 K → **不能**成笔 |
| **中继分型** | 笔未破坏前的新分型 → 延伸笔，不另起笔 |
| **弱顶/弱底** | 第三根未明显低于/高于 → 降低权重，等确认 |

---

## 3. 笔（成笔与破坏）

### 3.1 成笔条件（处理后 K 线）

| # | 条件 |
|---|------|
| 1 | 顶分型 + 底分型 **交替** |
| 2 | 顶底之间 **≥1 根独立 K**（顶底不共用） |
| 3 | 顶底分型间 **至少 4 根 K 线**（含顶底共 5 根，经典表述） |
| 4 | 后笔相对前笔：**向上笔** 顶更高且底更高（延续）或反向成新笔 |

### 3.2 笔的破坏

- **向上笔结束** ⇔ 出现**有效底分型**且其后形成**向下笔**（底低于前笔底或结构确认）。
- 笔破坏 ≠ 线段破坏；**多笔**才构成线段。

### 3.3 笔级别

| 操作级 | 笔 ≈ 时间跨度（经验） |
|--------|----------------------|
| 1min | 数分钟~数十分钟 |
| 30min | 0.5~2 天 |
| 日线 | 数周 |

> 随波动率变化；**以结构为准，不以时间为准**。

---

## 4. 线段（特征序列法）

### 4.1 为何需要线段

笔过多 → 噪音大。**线段** = 本级别走势的「折线」，由次级别**完整走势**构成。

### 4.2 特征序列

- 取**同向笔**的端点（向上线段取各笔高点序列 / 低点序列成对分析）。
- **特征序列出现分型** → 该方向线段**可能**结束。
- **反向线段**出现并破坏前线段极值 → 线段确认结束。

### 4.3 简化法（实战常用）

| 简化 | 规则 |
|------|------|
| **笔破坏 = 线段破坏** | 反向笔破前笔高/低 → 画线段转折 |
| 误差 | 中枢边界可能偏移 1~2 笔；精细复盘用标准法 |

### 4.4 线段级别

- 一段**日线线段**内，通常含 **≥1 个 30min 中枢**（级别联立经验）。

---

## 5. 中枢（定义与演化）

### 5.1 三段次级别构成

设某级别走势 a + b + c + …，其中 a、b、c 为**次级别走势**（各自由至少一线段）：

```
a 进入 → b 回拉 → c 再进 → 重叠区间 [ZD, ZG] 形成
```

| 符号 | 含义 |
|------|------|
| **a** | 进入段 |
| **b** | 反向段（不离开中枢） |
| **c** | 再次同向段 |
| **d、e…** | 延伸段（震荡继续） |

### 5.2 中枢区间

| | 算法 |
|---|------|
| **ZG** | max(各次级别重叠部分的**最低高点**) — 取重叠区上沿 |
| **ZD** | min(各次级别重叠部分的**最高低点**) — 取重叠区下沿 |
| 无效 | ZG < ZD → 无中枢，仅为次级别波动 |

### 5.3 延伸 · 扩张 · 新生

| 状态 | 条件 | 含义 |
|------|------|------|
| **延伸** | 中枢内 **≥9 段**次级别 | 仍在原中枢；不升级 |
| **扩张** | 后中枢 ZD~ZG 与前中枢重叠且满足升级定义 | 级别扩大 |
| **新生** | 后中枢区间与前**完全不重叠** | 趋势延续 |
| **趋势** | 上涨：后 ZD > 前 ZG | 两中枢同向且不接触 |

### 5.4 盘整

- 该级别**仅一个中枢**（或第二个未形成即反向）。
- 操作：**ZD~ZG 震荡**或等**离开**；勿按趋势加仓。

## 6. 走势递归表

| 本级别 | 次级别（约） | 本级别 1 线段 ≈ |
|--------|--------------|-----------------|
| 月线 | 周线 | 数月至一年 |
| 周线 | 日线 | 数周 |
| 日线 | 30min/60min | 数日至数周 |
| 60min | 15min/5min | 数小时至数日 |
| 30min | 5min/1min | 数小时 |

**递归律**：每个次级别走势本身 = 更低级别的「分型-笔-线段-中枢」完整实例。

---

## 7. 标准分解流程（盘后）

```
1. 选操作级 K 线 → 2. 包含处理 → 3. 标分型
→ 4. 画笔 → 5. 画线段 → 6. 标次级别并找三段重叠 → 7. 定 ZD/ZG
→ 8. 判盘整/趋势 → 9. 标记 a/b/c/d 段 → 10. 交 [[缠论-背驰与买卖点]]
```

---

## 8. 边界案例

| 案例 | 判法 |
|------|------|
| V 型反转 | 常缺标准中枢 → 按**小转大**或次级别递归 |
| 横盘数月 | 单中枢延伸 → 勿强行数趋势 |
| 跳空缺口 | 结构不断，力度比较慎用 MACD |
| 一字涨停 | 笔/线段可能无法细分 → 升级别操作 |
| 双底/W 底 | 可能是**两中枢**或**一中枢内两笔** → 看 ZD/ZG |
