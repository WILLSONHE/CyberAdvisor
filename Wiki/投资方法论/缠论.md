# 缠论（缠中说禅走势必完美）

> **定位**：A 股/期货通用的**结构分类 + 买卖点**体系；与 [[单根K线]]（单根语义）互补——缠论处理**走势结构递归**。  
> **原则**：**不测而测**（不赌路径，只判已完成结构）· **走势必完美**（任何级别走势终要完成）· **级别联立**（大级定方向，小级找介入）。  
> **保鲜**：方法论 → **永久** | 来源：缠中说禅公开课程与《教你炒股票》系列整理

*交叉：[[技术分析体系]] · [[单根K线]] · [[复盘与看盘]] · [[风控逻辑]] · [[仓位管理]]*

---

## 一、三大独立系统

| 系统 | 回答什么 | 核心产出 |
|------|----------|----------|
| **结构** | 现在走到哪一段 | 分型 → 笔 → 线段 → 中枢 → 走势类型 |
| **买卖点** | 在哪介入/退出 | 一买 / 二买 / 三买（空对称） |
| **背驰** | 力度是否衰竭 | 趋势背驰 / 盘整背驰 / 小转大 |

> 结构**可完全分类**；利润来自**级别与背驰判断**，非预测消息。

---

## 二、公理与哲学

| 命题 | 含义 | 操作含义 |
|------|------|----------|
| **走势必完美** | 任何级别的上涨/下跌/盘整终要完成 | 不猜「还有几浪」；等**完成信号**再转势 |
| **递归同构** | 同一套规则在 1min～月线重复 | 选定**操作级别**，上下各看一级 |
| **当下性** | 只依据已走出的 K 线 | 盘后或实时**包含处理**后再画结构 |
| **不测而测** | 不测目标价，测**结构完成度** | 计划：「若出现 X 结构则 Y 动作」 |

**走势** = 某级别上连续、不重叠的**线段**连接（由次级别走势构成）。  
**次级别** = 比当前低一级的完整走势（非简单 K 线根数）。

---

## 三、级别联立（实操框架）

| 角色 | 典型选法 | 用途 |
|------|----------|------|
| **操作级** | 30min / 60min / 日线 | 买卖点、仓位主依据 |
| **上级** | 操作级 ×4~6 时间 | 定多空：只在上级未完成下跌时做多 |
| **下级** | 操作级 ÷4~6 | 精确定位一买/二买、区间套 |

**纪律**：

- 不在**低于操作级**的波动里频繁交易（噪音）。
- 上级**趋势未结束**时，下级买点多为**短差**或**二买**，一买需上级也进入背驰段。
- **区间套**：在大级别背驰段内，用次级别重复「进入中枢 → 离开 → 背驰」找精确点。

---

## 四、K 线预处理：包含关系

**目的**：消除相邻 K 线高低点包含，使分型/笔定义唯一。

| 方向 | 包含规则（简） |
|------|----------------|
| **向上段** | 取高高、高低（向上合并） |
| **向下段** | 取低低、低高（向下合并） |
| **方向未定** | 前两根非包含 K 线定方向，再处理 |

**顺序**：按时间逐根处理；包含后的**虚拟 K 线**参与分型，不再看原始根数。

---

## 五、分型

| 类型 | 定义（包含处理后） | 意义 |
|------|-------------------|------|
| **顶分型** | 中间 K 线高点最高 | 局部卖压转折 |
| **底分型** | 中间 K 线低点最低 | 局部买盘转折 |

**有效分型**：左右至少各 1 根独立 K（顶底不共用同一根）。  
**延伸**：连续同向分型不都画笔——需等**反向分型**确认一笔终点。

---

## 六、笔

| 规则 | 内容 |
|------|------|
| **成笔** | 顶分型 ↔ 底分型，中间 ≥1 根独立 K（处理后） |
| **最少 K 线** | 顶底分型 + 中间 K → 通常 **5 根及以上**（含处理后） |
| **新笔** | 新顶高于前笔顶且新底高于前笔底（向上笔延续）或对称向下 |
| **笔破坏** | 反向笔成立 → 前笔在结构上结束 |

**注意**：笔是**直线近似**，忽略笔内小震荡；级别越低笔越多、成本越高。

---

## 七、线段

| 概念 | 定义 |
|------|------|
| **线段** | 至少 **3 笔** 同向叠加；由**特征序列**的分型破坏确认结束 |
| **特征序列** | 同向笔的端点序列；出现**顶/底分型** → 线段可能结束 |
| **线段破坏** | ① 被反向线段破坏 ② 特征序列分型确认（标准分解） |
| **标准 vs 简化** | 初学用**笔破坏 ≈ 线段破坏**简化；精细用特征序列 |

**级别**：线段是**本级别走势**的基本构件；一段上涨线段内可含多个次级别中枢。

---

## 八、中枢

| 项 | 定义 |
|----|------|
| **形成** | 某级别上 **≥3 段次级别走势** 的价格重叠区间 |
| **区间** | **ZD** = 重叠区最低 · **ZG** = 重叠区最高 · **中心** ≈ (ZD+ZG)/2 |
| **进入段 / 离开段** | 进入中枢的 a 段 · 离开向 d 段（可多次进出） |
| **延伸** | 在中枢内震荡 **≥9 段**次级别 → 延伸；不自动升级级别 |
| **扩张** | 后中枢 ZD~ZG 与前中枢重叠且更高一级定义满足 → 级别扩张 |
| **新生** | 前后中枢区间**完全不重叠** → 趋势延续 |

**盘整** = 该级别仅 **1 个中枢**（或未完成第二个）。  
**趋势** = 该级别 **≥2 个同向中枢**且**不重叠**（上涨趋势：后 ZD > 前 ZG）。

---

## 九、走势类型与完成

| 类型 | 结构 | 完成信号（简） |
|------|------|----------------|
| **上涨** | 离开中枢向上 + 可能再建中枢 | 出现**趋势背驰**或**更大级别盘整** |
| **下跌** | 对称 | 一买区或更大级别底 |
| **盘整** | 仅一个中枢内震荡 | 向上/向下**离开中枢**且**三买/三卖**或反向进入 |

**走势必完美**在此体现：未完成的上涨不会凭空变下跌——需**先**出现背驰或级别扩展/小转大。

## 十、背驰

### 10.1 两类背驰

| 类型 | 比较对象 | 典型场景 |
|------|----------|----------|
| **趋势背驰** | 同趋势中 **最后中枢前后两段**（c 段 vs 进入段） | 一买 / 一卖 |
| **盘整背驰** | 中枢内 **离开段 vs 进入段** | 盘整高抛低吸、二买辅助 |

**判断**：价格创新高/新低，**力度**（MACD 面积、黄白线高度、成交量）不创新高/低。

### 10.2 MACD 辅助（缠论标准辅助）

| 用法 | 规则 |
|------|------|
| **0 轴** | 黄白线在 0 轴下金叉 → 偏空区反弹；上穿 0 轴 → 多头段 |
| **柱面积** | 两段同向比较 **红/绿柱面积**；后段面积更小 → 背驰嫌疑 |
| **黄白线高度** | 价格新高而黄白线不新高 → 趋势背驰 |
| **盘整** | 中枢内比较进出段 MACD，不要求 0 轴 |

> MACD **不单独**定买卖点；必须落在**结构完成点**（一买/二买/三买）。

### 10.3 小转大

| 现象 | 处理 |
|------|------|
| 小级别背驰引发大级别反转，**无**标准大级别一买 | 立即降仓；不等大级别结构完美 |
| 政策/消息/缺口 | 可破坏标准背驰序列 | 结构优先，消息只做过滤 |

---

## 十一、三类买卖点（多为主，空对称）

| 类 | 定义（多） | 胜率/盈亏比 | 备注 |
|----|------------|-------------|------|
| **一买** | 下跌**趋势**最后一个中枢的**趋势背驰**低点 | 盈亏比高 · 易错 | 需趋势 + 背驰；非「跌多了就买」 |
| **二买** | 一买后**反弹**，再**回调不破一买低**的确认点 | 胜率较高 | 最常用；可叠加次级别一买 |
| **三买** | 中枢**向上离开**后，**回调不进中枢**（低点 > ZG） | 趋势跟随 | 假突破多；需上级趋势配合 |
| **一卖/二卖/三卖** | 对称 | | 空头或减仓 |

**不存在**「第四类」——三类已完备覆盖可介入结构。

### 11.1 与中枢位置关系（多）

```
下跌 → [中枢] → 一买（趋势背驰）→ 反弹 → 二买（不回中枢下沿）
              → 三买（离开中枢后回抽 ZG 之上）
```

### 11.2 盘整操作

| 策略 | 条件 |
|------|------|
| **中枢震荡** | 只在中枢 **[ZD, ZG]** 内 T+0 思维短差；破 ZD/ZG 止损 |
| **向上突破** | 离开 + 三买确认 → 转趋势思维 |
| **假突破** | 回拉进中枢 → 仍按盘整，不追 |

## 十二、分解与多义性

| 原则 | 说明 |
|------|------|
| **当下唯一** | 按规则当下只能一种**标准分解**；事后看可有多义 |
| **不预测** | 多义时**等走选择**后再跟，不提前押一种 |
| **结合律** | 大级别走势 = 小级别走势的**组合**，非简单相加 |
| **最低级别** | 软件上选能稳定成笔的最小周期；再小则噪音 |

**推荐流程**：定操作级 → 画中枢 → 标走势类型 → 找背驰 → 定买卖点类型 → 下级区间套确认。

## 十三、A 股实操约束

| 约束 | 缠论应对 |
|------|----------|
| **T+1** | 一买当日买，次日才能卖；**二买/三买**更适 T+1 |
| **涨跌停** | 结构可画，**力度比较失真**；等打开再确认 |
| **停牌** | 级别时间冻结；复牌后重画 |
| **指数 vs 个股** | 指数结构更稳；个股看**所属板块**同级结构 |
| **与 [[指数纪律框架]]** | 指数 L1–L5 纪律 = **上级方向过滤**；缠论买点须在纪律允许仓位内 |

## 十五、常见误区

| 误区 | 正解 |
|------|------|
| 数浪预测 | 只分类**已完成**结构 |
| 见底分型就买 | 可能是**笔内中继**，需级别 + 背驰 |
| 任意中枢都三买 | 三买需**趋势离开 + 不回 ZG** |
| MACD 金叉即买 | 须落在**一/二/三买结构** |
| 笔越画越多 | 级别升高或包含处理错误 |
| 忽略上级 | 小级二买可接大级下跌延续 |

## 十六、学习路径（压缩）

1. **包含 + 分型 + 笔**（1–2 周手工画 30min/日线各 20 例）  
2. **线段 + 中枢**（识别 ZD/ZG、盘整 vs 趋势）  
3. **MACD 面积**比较 + **一买/二买**复盘  
4. **三买 + 区间套** + 与 [[仓位管理]] 结合  
5. 实盘：**只做二买/三买** 直至一买误判率下降

## 十七、术语速查

| 术语 | 一行定义 |
|------|----------|
| 走势 | 某级别连续线段连接 |
| 次级别 | 构成当前结构的低一级完整走势 |
| 分型 | 顶/底局部极值 K 组合 |
| 笔 | 顶底分型间的最小结构单元 |
| 线段 | ≥3 笔 + 特征序列破坏 |
| 中枢 | ≥3 次级别走势重叠 [ZD,ZG] |
| 盘整 | 单中枢震荡 |
| 趋势 | 多中枢同向且不重叠 |
| 背驰 | 价创新而力度不创新 |
| 一买 | 趋势末背驰点 |
| 二买 | 一买后回调确认 |
| 三买 | 离开中枢后回抽不破 ZG |
| 区间套 | 大级别背驰段内用小级别找点 |
| 小转大 | 小级别背驰引发大级别反转 |
