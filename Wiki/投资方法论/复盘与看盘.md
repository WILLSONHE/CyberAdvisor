# 复盘与看盘

> 通达信版面 + 盘中监控 + 每日复盘 SOP。炒股需下功夫，**手搓笔记**优于只看 AI 摘要。

## 硬件与软件

| 项 | 建议 |
|----|------|
| **电脑** | 必要；手机信息密度与效率不足 |
| **软件** | 通达信（免费版基础功能足够；研究版约 4000–6000 元/年，4 月底有活动） |
| **屏幕** | 短线至少 **2–3 屏**；单大屏可四分屏（Wiki 5K 45 寸四分） |
| **消息** | **财联社**电报 + 语音播报 |

## 四屏版面（内容源默认）

```
┌─────────────┬─────────────┐
│  分析图/K线  │  上证分时    │
│  + 自选股    │  + 指数行情  │
├─────────────┼─────────────┤
│ 个股分时     │ 短线精灵     │
│ 关联报价     │ + 涨速榜     │
│ 可转债行情   │              │
└─────────────┴─────────────┘
```

### 各块用途

| 区域 | 内容 | 作用 |
|------|------|------|
| 分析图 | 个股 K 线 | 形态与趋势 |
| 自选股 | 核心池 | 快速切换 |
| 分时 | 个股 + 上证 | 盘中节奏 |
| 关联报价 | 同板块/产业链 | 点开即知「干什么」 |
| 可转债 | 可转债权 | T+0 联动练盘（**新手勿做**） |
| 短线精灵 | 异动监控 | 量化/大资金批量动 |
| 涨速榜 | 涨速前列 | 第一时间发现板块动 |

**看盘核心**：盯**监控精灵**——哪些票一起动 → 对应板块与龙头（需长期复盘积累熟悉度，约 **80–90%** 股票应能认出）。

### 列表字段

封单额、涨幅、成交额；配合全市场排序 + 精灵异动（如 13:37 批量 ETF 买入 = 资金打底信号）。

---

## 可转债联动（进阶）

- 给有转债的正股**批量标记** → 涨速榜见标记即知可 T+0 联动
- 例：正股冲板未买到 → 做对应转债（如斯特转债）；**仅作分时盘感练习，新手慎用**
- T+0 放大失误成本

---

## 双屏 / 第二软件

单屏只能看一只分时 → 另开窗口盯：

- **市场最高板**、集合竞价强弱（9:15 起）
- 9:30 爆量、开板、联动信号
- 持仓 / 北方华创 / 半导体锚定 / 京东方等容量票

期货才需六屏；股票 **钉几个关键票** 即可。

## 每日复盘 SOP

### 1. 涨停板

- 打开当日涨停列表 → 最强板块与龙头（例：京东方 → Micro LED / 光电批量涨停）
- 一字板看板块强度；回封 vs 炸板

### 2. 跌停板

- 连续跌停原因 → 回避走弱板块

### 3. 全市场金额排序

- 精力够则过 **300–500** 只：分时 + 形态
- 区分逆势强 / 板块内弱

### 4. 指数与情绪

- 涨跌家数、加速、情绪温度

### 5. 参考材料

- **涅槃重生** 77 页笔记（内容源短线进步最大来源）——对照「当日大盘 + 明日预判」手搓
- 淘股吧等复盘帖（付费/免费均有）；**新手须手搓**，AI 代劳反而弱化解盘力

**周期**：坚持 **3–6 个月** 每日数小时 → 市场感悟显著提升。

## 交易与生活平衡

| 习惯 | 说明 |
|------|------|
| 周五少答疑 | 周末休息、接人、放空 |
| 低脑力娱乐 | 无脑剧等，降低阈值 |
| 走出房间 | 公交、自然、换脑 |
| 睡眠 | 9:15 前清醒兴奋（需训练） |
| 提问时段 | 复盘发布后 **1–2 小时内**；盘中或 11 点后；避免深夜/早盘 |
| 随手交易 | 牛市尚可；**熊市/混沌期**回撤极大 |
| 镜子/摆件 | 下单前看状态，梳理情绪 |
| 长期主义 | 要**一直赢**，非赢一两次 |

有正职且通道清晰 → 可当副业，不必 All in 交易。
