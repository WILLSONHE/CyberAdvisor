## 1. 心态与纪律

| 原则 | 内容 |
|------|------|
| **不预测** | 只写 IF-THEN：「若出现 X 结构则 Y」 |
| **不喜好** | 结构>故事>「看好」 |
| **级别定仓** | 大级定方向，小级定时机 |
| **T+1** | 一买试探小仓；**二买主仓** |
| **指数过滤** | 上证结构 + [[指数纪律框架]] L1–L5 |

---

## 2. 标准操作程序（SOP）

```
盘前：定操作级(30/60min) + 上级(日线)
  → 包含处理 → 画笔/线段 → 标 ZD/ZG
  → 判盘整/趋势 → 标记买卖点类型
盘中：仅执行预案；小级区间套确认
盘后：更新结构；写次日 IF-THEN
```

---

## 3. 三类买点实战权重

| 买点 | 仓位 | 适用 |
|------|------|------|
| 一买 | 10–20% | 趋势末背驰；易错 |
| **二买** | **30–50%** | **主力**；回调不破前低 |
| 三买 | 20–40% | 趋势确认；须上级多 |

**止损**：破一买低 / 破 ZD / 三买破 ZG（机械）。

---

## 4. 盘整 vs 趋势

| 状态 | 策略 | 禁止 |
|------|------|------|
| 盘整 | ZD–ZG 内短差；等离开 | 按趋势重仓 |
| 趋势 | 二买/三买跟进 | 逆势一买 |
| 背驰段 | 分批+区间套 | 满仓抄底 |

---

## 5. MACD 实战

| 步骤 | 动作 |
|------|------|
| 1 | 定比较两段（同向、可比） |
| 2 | 数柱面积（0 轴下绿/上红） |
| 3 | 结构位 + 面积缩小 → 背驰 |
| 4 | 次级别再确认（区间套） |

陷阱：金叉/死叉 **alone** 非买卖点。

---

## 6. 失败与纠错

| 失败 | 应对 |
|------|------|
| 一买后继续新低 | 止损；等下一结构 |
| 假三买（回中枢） | 降仓；改盘整思维 |
| 小转大 | 不等大级完美，先降仓 |
| 消息/缺口 | 结构重画；力度比较暂停 |
