# 视频专题：商业航天的星辰大海 · SpaceX 万亿估值（2026-06-09）

> 2015 政策元年 → 2025 万星组网分水岭 → SpaceX IPO 全球估值锚 → 太空算力下一叙事

---

## 一、产业阶段：从航天工程到航天工业

| 阶段 | 特征 |
|------|------|
| **2015 元年** | 《民用空间基础设施 2015–2025》→ 民营资本可参与发射/制造/运营；长光「吉林一号」等验证 |
| **2015–2024** | 低频验证：发一星、验一型、拿一项目；看概念/远景 |
| **2025+** | **万星组网**、批量制造、规模运营；从实验 → **工业体系** |

> 逻辑切换：单星科研工程 → 千/万颗 **网络 + 持续交付 + 长期运营**。

## 三、占轨保频（ITU 规则）

- 低轨 **轨道 + 频谱** 稀缺；**先到先得**
- Starlink 在轨 **9600+** 颗；累计部署约 **1.19 万** 颗
- 中国在轨总量 ~**1200** 颗（通信 ~300；星网 ~160、千帆 ~162）
- 中国已向 ITU 申报约 **20 万颗** 规划；国内规划星座 **120+**、卫星 **6 万+**
- **部署比例**：2 年 **15%** / 5 年 **57%** / 7 年 **100%** → **只报不发则失效**

> 竞争本质：**容量 × 时延 × 覆盖 × 成本**；非单纯比数量。

| 拐点 | 要点 |
|------|------|

| **成本** | 可回收火箭 + 卫星批产；目标锚 **~200 $/kg** 入轨（SpaceX 公开拼车 ~**7000 $/kg** 增量；长期愿景 **10–20 $/kg**） |
| **需求** | 通信（全球）/ 遥感 / 卫星物联网 / **AI 边缘在轨**（天数天算） |

## 五、中美路径差异

| 维度 | 美国（SpaceX） | 中国 |
|------|----------------|------|
| 驱动 | 资本 + 超级公司垂直整合 | **战略自主** + 产业集群 |
| 地面禀赋 | 电力/土地/散热约束 → **太空叙事**（能源/空间套利） | 水电/核电/特高压/5G 强 → **更务实** |
| 算力路径 | 集中式、大规模轨道 AI 基础设施 | **小批量验证** → 天数天算 → 天地协同 → 再扩展 |
| 星座规模 | Starship 推大规模 | 首期多规划 **~216 颗**；千星可行、万星远期、百万级愿景 |
| 投资映射 | 单一平台 | **第二增长曲线** — 制造业龙头切入航天配套 |

**核心赛道（Wiki 判断）**：下一 10–30 年叙事主线 = **太空算力 / 太空数据中心**（非仅火箭/星网）。

### 太空算力四瓶颈

1. **能源**：ROSA 柔性太阳翼、钙钛矿/叠层电池  
2. **热控**：真空辐射散热；百 kW 级已难；百 MW 级需极大辐射面积  
3. **抗辐射计算**：宇航级 vs **商规+系统容错**（国内十余家半导体计划 **2026H2–2027H1** 在轨验证）  
4. **星间激光**：SpaceX **200G+**；国内 **100G 试验**阶段  

### 国内玩家类型（梳理，非推荐）

| 类型 | 代表 |
|------|------|
| 国家队/科研 | 之江实验室、北邮、中科天算；三体星座 **12 颗**整轨入轨 |
| 商业旗舰 | 国星宇航、轨道晨光（16MW 愿景）、东方天算 |
| 制造/配套 | 顺天谦（君达体系，1000 颗规划/50+ 颗订单）、热塑科技（热控/液冷） |
| 创业 | 太空字节（216 颗规划，与字节无关）等 |
