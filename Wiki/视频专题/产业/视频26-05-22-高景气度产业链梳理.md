# 视频专题：高景气度产业链梳理（2026-05-22）

> 大摩 VR200 拆解 → AI 机柜价值密度跃升 → PCB/MLCC/ABF 等下游补涨 → 半导体/MicroLED/光学平台延伸

---

## 产业链框架

### 1. 大摩 VR200 机柜价值结构（ODM 口径 ~780 万美元）

| 对比 | GB200 | GB300 | VR200 | VR200 vs GB300 |
|------|-------|-------|-------|----------------|
| 整机柜 | 333 万 $ | 399→700 万 $ | ~780 万 $ | **+95%** |
| GPU 占比 | 65% | 63% | **51%** | 非 GPU 组件涨更快 |
| 内存占比 | 7% | 9% | **26%** | 性能瓶颈核心 |

**逻辑**：AI 机柜从「CPU+内存+板载+电源」升级为 GPU/CPU/互联/HBM/高阶 PCB/液冷/机柜组装的**复杂硬件系统**；OEM 采购价高于 ODM。

### 2. 组件价值量变化（GB300 → VR200）

| 组件 | 涨幅 | 核心驱动 |
|------|------|----------|
| 内存 | **+435%** | HBM 需求；单机 37→200 万 $（已充分定价） |
| GPU | +57% | 252→396 万 $ |
| NVLink Switch | +122% | 6.48→14.4 万 $ |
| 网络芯片 | +121% | 26.1→57.6 万 $ |
| **PCB** | **+233%** | 层数 M7→M8、HDI 22→26 层、Switch 24→32 层、新增中配板/ConnectX/BlueField 模块 |
| **MLCC** | **+182%** | 高容高可靠、107 规格供给紧张 |
| **ABF 载板** | +82% | 单芯片载板价值翻倍、芯片数量增加 |
| 电源 | +32% | 120→200→380→600 kW；40V→400V HVDC 架构升级 |
| 液冷 | +12%（金额） | **渗透率标配化**（VR200 全液冷无风扇） |
| ODM 增值 | +38% | 14.96 万 $；复杂度↑ → 利润池扩张；2026Q2 或转**寄售模式** |

### 3. 传导链条

```
大摩 VR200 研报（5/20）
  → 内存已涨 → 资金下沉 PCB / MLCC / ABF（5/22 盘面）
  → 半导体设备材料零部件（业绩验证阶段）
  → MicroLED 光互联（新方向）+ 光测试仪器 + 精密光学平台
```

### 4. 半导体四层框架（简版）

| 层级 | 代表方向 |
|------|----------|
| 晶圆代工 | 中芯、华虹、燕东微 |
| 光刻供应链 | 茂莱光学、波长光电、福晶、汇成真空、洁美科技 |
| 前道设备 | 中微、北方华创、拓荆、盛美、华海清科、芯源微 |
| 后道封测设备 | 华峰测控、长川、金志达、金海通 |
| 材料（抗日） | 光刻胶（鼎龙、上海新阳、南大光电）、硅片（沪硅）、CMP（安集）、靶材（江丰） |
| 零部件 | 珂玛科技（陶瓷结构件）、富创精密、恒运昌、精仪装备、新莱应材 |

### 5. MicroLED 光互联（京东方×康宁 5/20）

- **范式**：数百并行低速通道替代少数高速通道（「宽而慢」）→ 低功耗（可至铜缆 5%）、高可靠、高带宽密度
- **阶段**：技术验证/早期导入 → 客户验证 → 小批量 → 良率/成本 → 订单兑现
- **微透镜阵列**：像素级耦合 → 通道数十→数百 → 透镜价值量结构性提升

## 受益方向

| 方向 | 逻辑 | 节奏 |
|------|------|------|
| **PCB / HDI / 高速 CCL** | VR200 层数+材料+新模块三重驱动；当日涨幅最快 | 研报催化后补涨 |
| **MLCC 涨价链** | 增量（AI 服务器成倍）+ 价升（高端紧平衡）+ 产能外溢 | 暗线→明线 |
| **ABF 载板** | 先进封装底座；Rubin Ultra 迭代继续提升 | 中长期 |
| **液冷** | 标配化；冷板/快接头/CDU/泵阀 | 等 Rubin 放量，中报或仍平淡 |
| **HVDC 电源** | 功率等级跃迁；连接器/铜排/电力电子 | 中长期 |
| **半导体设备零部件** | 概念→**业绩**；抗日替代 | 聚焦已释放业绩龙头 |
| **MicroLED 光互联** | 显示×光通信融合；数据中心短距互联 | 主题→订单前夜 |
| **光测试仪器** | 800G→1.6T→3.2T 测试难度跃升 | 国产示波器/测试设备 |
| **精密光学平台** | 光刻量检测 + AI 光互联（CPO/OCS/FSO/MicroLED）双曲线 | 平台型受益 |

## 标的线索（非推荐）

> 博主口径：理解「为什么涨」比追高位更重要；按池子+产业逻辑筛选，非买卖点建议。

### MLCC 三类

| 类型 | 线索 | 备注 |
|------|------|------|
| 成品龙头 | 三环集团、风华高科 | 跟随海外涨价、盈利结构改善 |
| 上游/零部件 | 昀冢科技、洁美科技 | 载带复苏 + 离型膜 + 艾弗斯光学设备收购 |
| 弹性/稀缺 | 力芯微 | 107 规格合作工序；摩尔线程供应商；业绩待验证 |

### 半导体（业绩优先）

- **已出业绩**：富创精密
- **重点跟踪**：珂玛科技（长存/长鑫/拓荆/华创客户；日本化）、芯源微（涂胶显影）、鼎龙/上海新阳/南大光电（光刻胶）
- **弹性**：恒运昌（射频电源）、精仪装备（温控+真空泵）、新莱应材（阀门管路）

### MicroLED / 光互联

| 环节 | 线索 |
|------|------|
| 芯片制造 | 华灿光电（京东方旗下；6 寸量产线；与芯相微合作） |
| 垂直延伸 | 兆驰股份（LED→光通信芯片/模块） |
| CIS 延伸 | 思特威（TX MicroLED + CMOS + RX 四类芯片） |
| 微纳光学 | 聚光科技（晶圆级微光学 + SMO 收购；OCS 小批量） |
| 光学元件 | 茂莱光学、水晶光电、蓝特光学、弘景光电 |

### 光测试

- 普源精电（示波器国产替代）、华峰测控、鼎阳科技、埃科光电（工业相机→光模块 AOI）

### 平台型光学

- **茂莱光学**：半导体光刻光学 + AI 光互联；可转债 5.63 亿扩产；远期规划半导体 20 亿美金 + 光互联 10 亿美金（听一听，风险自担）
