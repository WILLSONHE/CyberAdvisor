# 视频专题：单根 K 线 ② — 长黑线与影线（2026-06-06）

> 技术系列第 2 课 | 长黑三种情境 + 上吊/铁锤/流星/倒锤 | 5/14 高档长黑实战背景

---

## 工具 / Setup

| 项 | 说明 |
|----|------|
| **长黑标准** | 开盘≈最高、收盘≈最低；实体大 |
| **指数参照** | >1.5% 明显长黑；>3% 历史级；个股约 6–8%（指数 3% ≈ 批量跌停） |
| **与长红差异** | 长黑仅 **3 种**（无「长黑被跌破」— 长黑本身已是危险信号） |
| **影线逻辑** | 进攻**失败**的记录；失败有时比成功进攻信号更强 |
| **前置** | [[视频26-05-28-单根K线①]] — 长红线四情境 |

**纪律总纲**：**价格包含一切** — 不必盯消息；信息级别越大，价格反映越深。

## 步骤 / SOP

### 长黑线三种情境

#### 1. 高档长黑 → **无条件清仓**

- 上涨末端突发光头光脚长黑
- 原因（获利了结/利空/大盘崩塌）**不重要**
- 全天顺畅下跌、多头全线被套
- **不赌反弹**；5/14 上证、个股跌停式长黑、标普 500 同类
- 客观事实 > 主观判断（「基本面还在」等无效）

#### 2. 低档长黑 → 底部**反弹**信号（非反转）

- 下跌末端 + **非理性加速**：连续大阴、跌幅远超平时
- 叙事：最后一批散户/弱机构砍仓 → 卖盘极致 → 接近短期底
- **案例**：924 行情前；中际旭创 180→60（叙事未变，非理性杀跌）

**操作**：

- 仅**小仓**博弈反弹
- **严格止损**：破长黑最低点出；反弹至跌幅一半可止盈
- 不可满仓 all-in 当反转

**对照**：低档长红 = 多方主动反攻；低档长黑 = 空方力量枯竭。

#### 3. 箱体向下突破（翻箱向下）

- 长黑跌破箱体下沿 → 箱内多头被套 → 按箱体高度向下翻目标
- **向下突破不必须放量**；缩量下跌有时更危险（无人接盘）
- 放量跌破 = 看空一致 → 哪怕亏损也要离场
- 案例：金发科技熊市翻箱 23.19→18.47→目标 ~13.75

### 影线型 K 线

| 形态 | 位置 | 含义 | 次日确认 |
|------|------|------|----------|
| **上吊线** | 上涨末端 | 长下影（2–3×实体）；空方**首次**深打 — 多方控盘松动 | 需高开高走/长红化解；否则减 30–50% |
| **铁锤线** | 下跌末端 | 形态同上 | 偏多 |
| **流星线** | 上涨末端 | 长上影（2–3×实体）；多方进攻**最后尝试**结束 | 化解更难：需站上流星线最高点；先减 **50%** |
| **倒锤** | 下跌末端 | 形态同流星 | 多方**首次反击**，空方松动 |

**组合**：上吊 + 流星 → 「必死无疑」

**化解规则**：

- 上吊：次日高开高走 / 长红
- 流星：次日高开高走且**站上流星最高点**（更难）
- 当天可不走，**必须看次日**；K 线需下一根确认

**叙事对照**：

- 上吊 = 空方进攻**开始**
- 流星 = 多方进攻**衰竭**

## 纪律

| 场景 | 动作 |
|------|------|
| 高档长黑 | **无条件清仓**，任何主观理由无效 |
| 低档长黑 | 小仓 + 止损；反弹 ≠ 反转 |
| 箱体向下破 | 离场；看下一目标位能否站稳 |
| 上吊/流星 | 当日可观察，**次日**决定去留；流星减仓比例 > 上吊 |
| 顺势 | 顶部信号化解需超预期利好；当前环境难度高 |

### 5/14 语境（视频录制背景）

- 周五美股大跌 → A 股科技回调 = 计划内「挖黄金坑」
- 高档长黑后周一大概率跳空续跌；等下跌后再找**低档长黑**做多点
- 科技行情未结束（内容源看至 2027 底–2028）；当前是位置问题
