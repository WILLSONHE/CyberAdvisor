# 视频专题：如何设置盘面（2026-05-22）

> 通达信四屏版面 + 盘中监控 + 每日复盘 SOP + 交易与生活平衡

---

## 工具 / Setup

| 项 | 建议 |
|----|------|
| **硬件** | 电脑必要；短线至少 **2–3 屏**（Wiki 5K 45 寸四分屏） |
| **软件** | 通达信免费版够用；研究版 ~4000–6000 元/年（4 月底活动 ~4000+） |
| **消息** | **财联社**电报 + 语音播报 |
| **第二窗口** | 单屏只能看一只分时 → 另开软件盯最高板/持仓/锚定票 |
| **L2** | 盘口深度按需付费 |
| **参考社区** | 淘股吧复盘帖；**涅槃重生 77 页**（内容源短线进步最大来源） |

### 四屏默认版面

```
┌─────────────┬─────────────┐
│  分析图/K线  │  上证分时    │
│  + 自选股    │  + 指数行情  │
├─────────────┼─────────────┤
│ 个股分时     │ 短线精灵     │
│ 关联报价     │ + 涨速榜     │
│ 可转债行情   │              │
└─────────────┴─────────────┘
```

| 区域 | 作用 |
|------|------|
| 分析图 + 自选 | 形态切换、核心池 |
| 上证分时 + 指数 | 大盘节奏 |
| 关联报价 | 点票即知板块/产业链 |
| 可转债 | T+0 联动（**新手勿做**） |
| 短线精灵 | 批量异动 → 量化/大资金 |
| 涨速榜 | 第一时间发现板块动 |

**列表字段**：封单额、涨幅、成交额；全市场排序 + 精灵异动（如 13:37 批量 ETF 买入 = 资金打底）。

### 可转债标记（进阶）

- 有转债正股**批量标记** → 涨速榜见标记知可 T+0
- 正股冲板未买到 → 做对应转债（如斯特转债）；仅作分时盘感练习

### 版面导入

Wiki 动态发放版面文件 → 通达信 **版面管理器 → 导入**。

## 步骤 / SOP

### 盘中看盘

1. **盯监控精灵** — 哪些票一起动 → 对应板块与龙头（需长期积累，~80–90% 票应能认出）
2. **涨速榜 + 板块涨速** — 强势/逆势/砸盘板块
3. **关联报价** — 陌生票一键看「干什么」
4. **第二屏分时** — 最高板、集合竞价（9:15 起）、9:30 爆量/开板、持仓与锚定票（北方华创、南芯、京东方等）
5. **财联社** — 重要消息推送

### 每日复盘（建议手搓）

| 步骤 | 内容 |
|------|------|
| 1. 涨停板 | 最强板块与龙头（例：京东方 → Micro LED 批量涨停）；一字板看板块强度 |
| 2. 跌停板 | 连续跌停原因 → 回避走弱方向 |
| 3. 全市场金额排序 | 精力够则过 **300–500** 只：分时 + 形态 |
| 4. 指数与情绪 | 涨跌家数、加速 |
| 5. 对照涅槃重生 | 当日大盘 + 明日预判，手搓笔记 |

**周期**：坚持 **3–6 个月** 每日数小时 → 市场感悟显著提升。

**原则**：新手须**手搓**；AI 代劳弱化解盘力（像看答案 vs 自己做题）。

## 纪律

| 场景 | 规则 |
|------|------|
| **随手交易** | 牛市尚可；**熊市/混沌期**回撤极大 |
| **下单前** | 镜子/摆件 — 检查情绪状态 |
| **长期目标** | 要**一直赢**，非赢一两次 |
| **提问时段** | 复盘发布后 **1–2 小时**；盘中或 11 点后；避免深夜/早盘（9:15 盯盘） |
| **周五** | 内容源少答疑，周末休息 |
| **正职清晰者** | 可当副业，不必 All in |

### 交易与生活

- 低脑力娱乐降阈值；走出房间（公交、自然）；睡眠保证 9:15 前清醒
- 炒股刺激拉高阈值 → 需主动平衡，避免神经长期紧绷
