## 步骤 / SOP

### 1. 大小周期仓位管理（核心）

| 原则 | 说明 |
|------|------|
| **大赚在主升** | 4/8 满仓、4/16 算力主线等节点 |
| **少亏在调整** | 5/14 日线顶 → 最快 ~16 交易日；当前约调整一半 |
| **时间 vs 空间** | 二者确定其一即可；小级别周期 + 情绪推断时间点 |
| **不猜空间** | 例：5/26 下午 02:05 反弹时间可推，幅度超预期 |

**历史节点（博主节奏）**：

- 12/17 抄底 → 2 月主升
- 2/26 减仓 → 规避 3 月黑天鹅
- 3/23 至暗时刻抄底
- 4/8 确认主升、4/16 算力租赁主线满仓

### 2. 抄底三信号（5/26 实例）

1. 新股/次新股表现（如 01:30 次新拉抬 — **勿新手追次新**）
2. 5 分钟级别顶部叠加 → 调整周期 ~2 小时
3. 情绪判断叠加 → 提高时间点准确度

### 3. 每日 10 分钟最小集

1. 读**复盘** + **早盘**（各一遍）
2. 周日加**周复盘**
3. 盘中扫**动态**（有无仓位/节点提示）
4. 节假日集中整理充电合集

### 4. 产业逻辑跟踪

- 4–11 月主线：**机构产业线戴维斯双击**
- 半导体 = **多年超大周期**（类比比亚迪中间 50% 回调仍长期向上）
- 5/26 韬定律：半导体从「长鑫扩产」→ **全产业链**；华大九天等 EDA 启动

### 5. 提问与互动

- **不问买卖点**；个股问题放评论区（其他学员可能更熟）
- 有效提问窗口：复盘后 1 小时内或盘中
- 涨价信号：博主涨价 ≈ 主升浪临近（当前大周期不好暂不涨）

## 纪律

| 规则 | 说明 |
|------|------|
| **跟节奏** | 相信节点提示，勿在差周期重仓 |
| **不追次新** | 抄底信号含次新 ≠ 让散户买次新 |
| **勿读不懂的机构原文** | 卖方资讯看不懂就买 = 套；等博主提炼 |
| **少亏大赚** | 下跌控仓 + 上涨加满 → 散户可持续盈利 |
| **长期主义** | 亏 90% 仍有机会；关键是未放弃 + 跟体系 |
| **胡思乱想有害** | 博主取辅助作用，减少恐惧贪婪下的伤害性行为 |

- 午后拉升超预期 → **明天决战**（能否真走强）
