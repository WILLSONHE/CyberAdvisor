# 视频专题：单根 K 线 ① — 长红线（2026-05-28）

> 技术系列第 1 课 | 单根 K 线 → 组合 K 线 → 形态 → 趋势（约 4–6 个月）| 磨底/震荡期权重最高

---

## 工具 / Setup

| 项 | 说明 |
|----|------|
| **适用市场** | 非明确主升/熊市；**反弹与磨底**阶段 K 线最有用 |
| **K 线基础** | 阳线 = 收 > 开；下影线 = 最低价；上影线 = 最高价 |
| **涨跌幅参照** | 指数 >1% 明显、>3% 很强、>5% 历史级；个股约 ×3–5（大盘 3% ≈ 个股涨停） |
| **学习路径** | 普通人最适合**技术分析**（vs 价值投资需调研、情绪周期需多次破产） |
| **前置** | [[第四课有效市场假说]] — 为何现在技术仍有用但需学深 |

**光头光脚阳线（长红线）三特征**：开盘≈最低、收盘≈最高、实体大（不必几何完美）。

---

## 步骤 / SOP

### 核心原则：位置决定含义

同一根长红线 — 低档 = 买；高档 = 卖。忌「手里锤子看什么都是钉子」。

---

### 情境一：低档长红（多方启动）

**定义**：较长/较深下跌后突发标准长红。

**四条件**：

| # | 条件 |
|---|------|
| 1 | 真实下跌 **5–10 交易日+**，越长越好 |
| 2 | 跌幅够深：个股 ~**15%**，指数 ~**5%** |
| 3 | 突发标准长红，实体显著 |
| 4 | **放量**强信号；恐慌底可缩量仍有效 |

**叙事**：卖压耗尽 → 大资金一日吃尽卖盘拉至最高 → **短期底部**（涨多久不猜）。

**案例**：茅台 2026-02 长期阴跌后长红（买） vs 已涨一段后的长红（卖）。

---

### 情境二：高档长红（多方耗尽）

**特征**：长期温和小阳后，突发**远超以往**的光头光脚大阳，可带**跳空**。

**逻辑**：最后一批观望/犹豫资金入场 → 买盘当日耗尽 → 多为**顶**。

**口诀**：「不见长虹不回头」— 跳空光头阳常为顶。

**操作**：不追；**越涨越卖**（非一次性清仓，可能还有最后一波）。

**案例**：茅台 2400 后再未新高；中芯国际上涨后类似形态。

---

### 情境三：箱体向上突破

- 箱体内拉锯数周/月 → **放量**长红突破上沿 = 偏多
- **必须放量**；缩量突破易假突破跌回
- **目标参考**：箱体高度翻一倍（小箱/大箱分别估，作计划非精确点位）
- 到目标位可分批出；突破后收难看阴线应离场

**翻箱示例（中芯国际）**：小箱 84–94 → 目标 ~104；大箱可算 ~141。

---

### 情境四：长红被跌破（风控）

长红 = 当日买入者成本区 = 支撑。

| 防线 | 动作 |
|------|------|
| 实体**中点** | 警戒，**先减仓** |
| 实体**最低点** | **无条件清仓** |

无纪律的技术分析 = 赌博。

---

## 纪律

1. **位置优先** — 先判档位再判形态
2. **放量确认** — 向上突破必看量；跌破长红必执行减仓/清仓
3. **有计划交易** — 翻箱目标价变模糊感觉为清晰计划
4. **人形量化** — 学技术 = 严格纪律，长期存活
5. **系列节奏** — 本课仅长红线；长黑线见 [[视频26-06-05-单根K线②]]
