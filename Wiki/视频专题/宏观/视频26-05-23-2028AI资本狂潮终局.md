# 2028：AI 资本狂潮的终局推演

> B站 BV1gkGq6pEuG | 2026-05-23

---

## 核心框架

**起点**：大摩拆解英伟达 VR200 机柜 **780 万美元**（约 5300 万人民币）——内存成本较上一代 +435%，是「警报」而非单纯胜利。

**第一层认知：金融危机中的「钱去哪了」**
- **钱守恒，价格不守恒**：新闻「蒸发 X 万亿」多为边际共识修正（幻觉），非真实货币消失
- **五类真实转移**：高位卖出者←高位接盘者；逆周期买入者←被迫卖出者；借款人←存款人（通胀）；救助接收者←纳税人；机构←恐慌散户
- **短寿命资本品**：GPU 物理寿命 6–8 年、经济寿命 3–5 年——大量支出是**消费 disguised as 资产**；价值毁灭真实，但现金已流入产业链创造就业/GDP

**第二层认知：经济是流量而非存量**
- 美国 2025 年 GDP 边际贡献中 **AI 资本开支占 30%–40%**；若无 AI  capex，增速或 <1%
- 1500–2000 万人直接或间接依赖 AI 资本开支持续流动
- 危机本质：**不是过去的钱消失，而是未来预期被打碎**——所有基于「流量持续」的决策被迫修正

**佩雷斯技术革命框架**：导入期（20–30 年金融资本涌入→泡沫→大型金融危机）→ 转折时刻 → 展开期（技术渗透全社会）。五次革命转折：1797 / 1847 / 1893 / 1929 / 2000——**AI 对应下一次约 2028 前后**。

**AI 泡沫特殊结构**：
- **摩尔定律失效**：2nm 后性能提升放缓、成本上升 → 英伟达靠堆料（VR200 400 万→780 万）
- **GPU 四层抵押品化**：CoreWeave 等租赁公司、云厂商 AI 债券、OpenAI 结构化债务、主权/私募 AI 基金——类似 07 年 MBS，但 GPU 是**短寿命抵押品**（ unprecedented）
- **DeepSeek 时刻**：557 万美元训练 vs OpenAI 数亿美元——捅穿「AI 必须极贵」叙事；推理价格塌方（$30/M token → $1）
- **美国电力瓶颈**：数据中心耗电 +233% vs 装机 +18%；变电站 7–10 年；燃气轮机订单至 2030 后
- **中国优势**：算电协同、特高压、东数西算；国产芯片（昇腾、摩尔线程等）；EUV 追赶窗口（摩尔定律失效→前沿放缓）

## 关键论证

### 危机传导机制（AI capex -50% 假设）

1. **产业链收缩**：英伟达、台积电、HBM、设备商订单砍半
2. **就业恶化**：数据中心建设、电力设备裁员；GPU 融资公司破产
3. **消费急刹车**：失业→削减非必需消费→餐厅、房产中介等二级传导
4. **乘数反转**：1 美元 capex 削减 → 2–3 美元 GDP 收缩
5. **信心崩塌 + 债务螺旋 + 资产负债表衰退 + 金融体系瘫痪**（雷曼剧本）

**张先生案例**：40 岁、年薪 100 万、高杠杆生活决策——失业后真实损失是**人生轨迹破产**，非账面数字。

### 2028 前后时间表（Wiki推演）

| 时点 | 事件 |
|------|------|
| 2026 Q3–Q4 | 大模型收入增速放缓；云厂商下调 2027 capex 指引 |
| 2026 末–2027 Q1 | 台积电/HBM 产能落地；H100 二手价首次回落（2.5 万→1.8 万美元） |
| 2027 Q2–Q3 | CoreWeave 利用率降、债券利差扩大 |
| 2027 Q4–2028 Q1 | CoreWeave 等破产保护——**雷曼时刻** |
| 2028 Q2–Q3 | 云厂商 GPU 折旧 6 年→3 年；利润 -30%–40%；H100 二手 ~8000 美元 |
| 2028 Q4 | 美联储降息+QE；分化反弹——应用层、中国国产替代、黄金 |
| 2029–2030 | 新秩序：中国算力/电力/制造优势兑现；美国 AI 从霸主→高端利基 |

### 危机后的 paradox

- VR200 从 780 万→80 万 → **算力白菜价** → AI 普及、机器人经济模型打通
- 1929 电气化、2000 光纤泡沫后——基础设施廉价化才开启真正展开期
- **2030 年代主角**：可能不在硅谷，而在深圳/杭州/县城医院/非洲乡村——便宜硬件+电力+开源模型

### 七项监测信号

1. 台积电 CoWoS 利用率 <90%
2. HBM 现货/合约价比 <0.85
3. 英伟达收入同比下滑
4. 云厂商 2028 capex 指引增速 <15%
5. CoreWeave 债券利差 >500bps
6. H100 二手价 <1.8 万美元
7. 中国 EUV 原型机小批量验证

## 对 A 股/资产含义

| 资产 | 判断 |
|------|------|
| **中国资产** | 产业霸权转移的相对受益方；危机后修复更快；**不代表 2026 立刻涨** |
| **国产算力/电力/关键矿物/机器人** | 2029–2030 优势全面兑现 |
| **AI 硬件（短期）** | 2026–2028 仍处导入期高位，对外部冲击敏感 |
| **黄金** | 危机深度阶段或暴涨 |
| **应用层/普惠 AI** | 危机后算力廉价化受益 |

**个人策略**：
- 固定支出基于**长期可持续收入**，非巅峰收入（100 万年薪→固定支出 ≤50–60 万）
- AI 产业链从业者：18–24 个月现金缓冲、可迁移技能、杠杆 < 收入 50%
- 流量思维：多元化现金流，非仅看存量资产
- 反脆弱行业：医疗、教育、必需消费、水电、农业——AI 繁荣期「无聊」行业或是避风港

## 风险与情景

| 情景 | 说明 |
|------|------|
| **基准（2028 危机）** | AI capex ROI 算不回来 + GPU 抵押品链断裂 + 摩尔定律失效 + 中国替代/电力瓶颈 |
| **延迟 1–2 年** | 美国对外冲突（伊朗 AI 无人机等）消化过剩产能，延长泡沫 |
| **低烈度地缘** | 台海、菲日摩擦、技术封锁升级——不全面战争但波动增加 |
| **生产率证伪** | 若 AI 10 年仅 +0.06% TFP（阿西莫格鲁）vs 高盛 1.5%——决定危机深度 |
| **危机后** | 算力白菜价→机器人/AI 普及；中国主导展开期 |

**核心公式**：金融危机真实转移 = 高位卖出 + 逆周期买入 + 借款人通胀收益 − 现金持有人损失 − 恐慌散户损失 − 纳税人救助成本
