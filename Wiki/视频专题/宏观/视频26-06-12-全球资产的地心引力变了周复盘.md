# 全球资产的"地心引力"变了（周复盘）

> 青枫浦上Q · 2026-06-12 · 全球定价锚切换：美伊/原油/沃什首秀 + AI 调整第一周 + A 股电风扇至 7 月会议

## 核心观点

- **全球资产「地心引力」在变**：美伊协议预期、原油、下周三 **沃什** 美联储首秀与 AI 估值再定价，共同拉扯风险资产。
- **美股今晚关键**：Oracle 发债 + SpaceX 超千亿认购持续从 **七巨头** 抽离资金；若滞涨且量能萎缩 → 主力不愿用真金白银赌周末签字 → 下周 A 股或再探底踩一脚。
- **AI 调整才第一周**：英伟达自高点约 **-15%**，不宜因短线反弹断言「泡沫已破」；未来 1–2 年重点盯 **AI ROI**。
- **A 股牛市中段已走完**：2025 年「拔估值」后好资产不便宜；6–7 月大概率 **电风扇 + 小碎步**，硬科技调整看 **1–2 个月**。
- **被动投资者**：**保险** 可躺（股息高、标的少）至 **7 月国内会议**；券商套牢盘重，只宜波段。

## 全球宏观与美联储（沃什首秀）

| 维度 | 判断 |
|------|------|
| **6 月 FOMC** | 利率不变近乎确定；焦点在 **点阵图** — 3 月仍有人预期降息，6 月或不降息阵营扩大甚至现 **加息讨论** |
| **沃什困境** | 鸽 → 被解读扛不住政治压力放任通胀 → 长债遭抛售、长端利率飙升反噬实体；鹰 → 风险资产承压 |
| **可能话术** | 多谈 **体制变革**、弱化/取消点阵图，少给利率承诺，用技术议题分散「加不加息」注意力 |
| **VIX** | 若声明删除「下次可能降息」暗示 → VIX 易飙升 |
| **原油** | 已跌破 **90 美元**；续跌利好风险资产，但难抵消 FOMC 前不确定性 |

## 美股、AI 与全球分化

- **vs 2000 互联网泡沫**：2021 极端更烈；当下少数超级公司撑指数、多数原地踏步（类似 A 股），上涨靠 **盈利预测** 而非纯讲故事 — 距泡沫不远但尚未等同 2000/2021。
- **英伟达信号**：业绩好但涨完又跌 → 难带动板块 = **波动率上升**；仍会有机构押 **存储、云巨头、光、半导体、AI 电力、液冷** 等物理配套。
- **全球分化**：防守 = 存储 + 云巨头；进攻弹性 = 光 / 液冷 / 电网设备。风险偏好降后选股难度显著上升，同板块内走势天差地别。
- **长期推演（2027–2028）**：若 A 股半导体营收/出货爆发 → 或冲击海外 AI 半导体巨头 → 戳破美股 AI 泡沫的潜在触发点；**长鑫上市** 后存储景气 → 「运力→存力→算力」链条。

## A 股盘面与板块

| 主题 | 要点 |
|------|------|
| **科技调整** | 重点观察池约 **80% 近端下跌**；平均调整 ~10%、最深 ~15%；二线光（如天孚通信）跌幅更大 — **科技已在调整，勿以个别反包代替整体** |
| **中际旭创** | 500 亿+ 阴线量能否「贪」需问号；可参考 **中天科技** 射击之星后次日反包 — 个案可化解，不能代表板块 |
| **时间尺度** | 调整 **1–2 个月**；参考中际 2024/12–2025/4 横盘而光模块等业绩期才表现 |
| **指数** | 全 A 等权调整极深 → 指数可强；新缺口出现但 **历史缺口大概率回补** |
| **6–7 月节奏** | 上证或小碎步上，难现持续主升浪；等新主题（如去年稳定币级）否则 **电风扇** |
| **配置** | 无交易能力 → **保险**；有能力 → 抓美股反弹节点做波段（如 6/11 周五） |
| **黄金** | 4230 跌破仍偏多短；**反弹非反转** — 3900–4300 合理区；Fed 若出问题看 **3500–3800**；降息周期远期仍可看 5000+ |
| **有色** | 超跌反弹，捞到就走 |
| **方向** | 硬科技（尤其北美相关液冷/燃机）风险仍大、调整未结束；**国产替代/光刻机** 可跟；商业航天、创新药、机器人（弱关联硬科技）可看；风电等看 **宁德时代** 企稳 |
| **港股** | 仍左侧，无新变化 |

## 操作纪律

- 区分 **调整已结束** vs **仍须调整** 的标的 — 决定持仓风格。
- 电风扇行情：**该走就走**，勿死赖单一方向。
- 一切未明前 **控风险**；最迟 **7 月** 宏观与国内会议或给出更清晰答案。
