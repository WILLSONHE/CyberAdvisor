# 倒车接人

> 无实质利空下的指数错杀，主升向震荡过渡；非全面退潮，震荡预期下买阴优于追涨。

## 核心观点

| 维度 | 判断 |
|------|------|
| 定性 | 日线高九被砸失效，属**明显错杀**；强主升 → **震荡过渡**，类比标普500加速后横盘再上 |
| 减仓逻辑 | 量能约3.3万亿、未见接近4万亿前风险可控；指数不会A杀，故非减仓点 |
| 外部因素 | 官媒「千元股高处不胜寒」= 温度调节器；大资金预判周五减仓而周四提前砸 |
| 地缘/会谈 | 合作与贸易已谈妥；**霍尔木兹海峡**及美方表态两项仍无定，结论前观望 |
| 活口分类 | 须区分**退潮活口**（掩护撤退）与**复苏活口**（下一波力量） |
| 长期框架 | 人民币破七、QE与年中行情逻辑未变，调整完仍看后续上行 |

## 指数/盘面判断

- 昨日情绪极度亢奋，今日高开下杀；算力租赁低于预期，但大周期**无法看空**
- 主线首次出现明显亏钱效应（工业富联），行情进入**震荡分歧**，非金融全面接力
- 尾盘涨跌家数未达中午低点，**未到二次冰点**；11:20与尾盘介入性价比高
- **创业板**明后日能否有力修复是关键：修复弱则硬件需减仓；修复强则视程度加减仓

## 板块与标的

| 方向 | 要点 |
|------|------|
| 国产算力 | 中国长城为CPU锚，踩10日线合理；中科曙光、润建股份等核心继续关注 |
| 科技修复 | 工业富联须止跌；算力租赁/国产算力能否主动领涨 |
| 低位切换 | 猪肉、电力、油气、光伏、机器人等能否走出持续性而非一日轮动 |
| 防守观察 | 银行、保险介入不深 → 资金**未放弃科技** |
| 风险票 | 跌停个股需警惕；除中国长城外其他票适当控险 |

## 操作纪律

- 对未来行情保持**震荡预期**：下跌敢买，主升式追涨停用
- 节点不清晰不出手；科技只看最核心标的
- 非科技方向看**趋势加持与持续性**
- 明日三观察：①工业富联止跌+科技主动领涨 ②低位切换持续性 ③是否杀出更充分冰点
