# 大盘将迎来反弹

> A浪下杀近尾声，B浪反弹目标看4200；吃完B浪再减仓，6月中下旬C浪结束后接黄金坑。

## 核心观点

| 维度 | 判断 |
|------|------|
| 波浪定位 | 上证经历A浪下杀，进入**B浪反弹**；B浪一般不超过前高，目标约**4200** |
| C浪展望 | C浪低点或破4100/**4080**，时间约**6月中下旬**；结束后为较好黄金坑 |
| 减仓节奏 | 当前点位不减仓，**B浪吃完再降仓**，C浪底部再接主线 |
| 拐点信号 | 周五W底 + 今日第二脚W底，双底成功率高于单底；英特尔盘前上涨强化拐点判断 |
| 下杀性质 | 今日下杀「不流畅」，区别于见顶阴线的流畅杀跌 → 更像中继而非见顶 |
| 空间预期 | B浪指数空间约**3%不到**，个股弹性更大 |

## 指数/盘面判断

- 每一轮下跌均有支撑；出现反转信号即迎上涨
- 若反弹至4200放量突破前高，则ABC调整剧本作废
- 反弹强度弱则拉长时间，强则压缩时间；站回**4180**可多留意数日
- 早一天晚一天冰火两重天：拐点须以当日最低点成立为前提

## 板块与标的

| 方向 | 要点 |
|------|------|
| **长鑫存储** | B浪最可能主线；低开为买点，观察是否跟随大盘反弹 |
| 算力租赁 | 三大运营商加速未封板 → 润建股份等客户链承压 |
| 半导体 | 5/20英伟达FY2027 Q1 + 英特尔止跌企稳 → A股不必过度悲观 |
| 备选方向 | 软件、中科曙光、中国长城（绑英特尔走势，明日阳线概率大） |
| 防守板块 | B浪反弹时承压；C浪下跌过程中或有一定利润 |

## 操作纪律

- B浪反弹遵循**买阴不买阳**，不追高
- 长鑫跟随B浪走完后该减仓减仓、该割肉割肉
- 不做长鑫者观察自身标的能否被反弹带动；板块动个股不动需个案分析
- 若指数直接A杀则需另找原因，但当前认为可能性不大
