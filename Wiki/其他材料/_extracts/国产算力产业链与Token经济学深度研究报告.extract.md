投资研究 · 行业深度

2026 年 5 月

一、报告摘要与核心观点

1.1 半导体超级周期外溢的中国时刻

自 2023 年 ChatGPT 引爆 AI 算力需求以来,以英伟达为代表的美股算力芯片公司股价持续上涨,延续 3 年的超级周期已传导至大空头开始做空美股芯片股的阶段。中国半导体板块此前反应相对滞后,核心原因在于两大困境:其一,先进制程产能跟不上;其二,生态不成熟,有限产出难以被市场接纳。

DeepSeek V4 版本发布之后,这两端瓶颈同步突破。其在发布信息中明确披露:"受限于高端算力,目前 Pro 数据吞吐量非常有限,等到下半年昇腾 950 超节点批量上市之后,Pro 价格能够大幅下调。"这段表述包含三个重要关键词——超节点、价格下调、批量上市,共同构成国产算力闭环跑通的标志性信号。

超节点:通过先进互联架构将成百上千张卡组成新的计算单元,以集群层面的协同弥补单卡性能的差距。华为 384 超节点(基于 910C)的整体性能相当于英伟达 NVL72 的 1.7 倍,但需要 384 张卡;到 950 时代对标英伟达 NVL144 超节点,所需卡数高达 8192 张,对芯片产能形成指数级拉动。

价格下调:国产算力将能够实现成本控制,使国产大模型相对海外模型具备更明显的性价比优势,从而打通需求侧逻辑——企业有理由采购国产卡。

批量上市:相当于向全市场宣告中国半导体制造链条已基本走通,产能可保证、交付可稳定,客户可以放心下订单。

二、国产算力闭环生态:从卡点突破到全链路放量

2.1 华为昇腾 950 超节点:产业放量的关键引擎

华为昇腾 950 超节点是本轮国产算力产业突破的核心载体。从产业放量的卡点来看,950 的产能要求具有两大特征:

先进制造 + 先进封装协同推进:昇腾 950 是国产先进设备的第一大终端客户,带动国产光刻胶、靶材、半导体设备(前道、后道)、半导体材料同步迎来超预期业绩兑现。

高度自主可控:在美国持续收紧出口管制(包括最严芯片法案)的背景下,950 必须实现关键环节全国产化,推动光刻设备、刻蚀、薄膜沉积、量测等环节的国产替代加速。

4 月底已披露的年报和一季报印证了上述判断:算力租赁公司业绩超预期、半导体材料(包括靶材)业绩集体超预期、先进制程扩产带动的前道设备景气度上行,后道封装测试环节同样强势。光刻机、光刻胶在最严芯片法案后从"刻"的逻辑延伸至"光"的逻辑,产业链景气度持续向上验证。

2.2 受益环节梳理:超节点放量的产业链全景

950 超节点采用全液冷散热、高速背板连接器和全光互联三大基础设施技术,这些原本主要给海外巨头供货的赛道公司,现在迎来确定性的国内增量订单。

2.3 科创板的预期差:本轮真正受益的核心标的

23 至 26 年间,A 股 AI 板块上涨的公司主要是给英伟达、谷歌、亚马逊供货的海外链厂商。真正国内自主可控路径下的科创板公司此前相对冷清——截至本轮上涨之前,科创 50 尚未回到 2020 年的高点。但随着超级周期向国内传导,科创板有望迎来从"被低估"到"价值重估"的窗口。

市场重估的核心逻辑包括:整个国产链条 AI 基础设施的价值重估、算力卡从"能用"到"大用"、从"稀缺"到"放量"的产业化跨越、从 10 到 100 的商业落地。

三、Token 工厂模式:算力商业模式的范式跃迁

3.1 概念界定:算力租赁、Token 分成、Token 工厂的关系

Token 工厂是 2025 年下半年至 2026 年开始在产业内浮现的全新商业形态,其本质是算力租赁向价值附加生意的商业模式升级。理解这一模式需要先明确三个相关概念的差异。

3.2 Token 工厂模式的盈利能力测算

基于产业调研数据,Token 工厂模式相比传统算力租赁的盈利能力提升来源于三个方面:

Token 调用分成的杠杆效应:服务器折旧与电力消耗属于刚性固定成本,Token 调用带来的超额分成几乎全部转化为纯利润,为算力租赁服务环节打开了极高的利润弹性。

算力使用率的精细化运营:Token 工厂通过算力细切割按需收费,白天用户 / 晚上用户、3 小时 / 5 小时、复杂任务 / 简单任务可以错峰使用,提升设备利用率。当前测算基于每日有效使用时长 8 小时,如果提升至 12 小时,算力使用效率还能再提升 50%。

价格机制的灵活性:当前 Token 工厂收费标准为市场主流大模型对外报价的 7 折左右(招商引资、互惠合作)。未来即使 Token 价格下跌,有效时长提升也可抵充价格下降。

根据机构调研测算,Token 工厂模式正常情况下两年有余即可完全收回成本,远优于传统算力租赁 5 年回本的周期。头部算力租赁运营商净利率有望从此前的 10-15% 强势逼近 30% 甚至以上。

3.3 Token 工厂的兴起背景:供需两端的共同推动

从需求端看,豆包(月活 3.45 亿)等头部应用进入 C 端付费阶段,标准版 68 元、加强版 200 元、专业版 500 元的分层订阅模式既验证了 C 端付费意愿,也直接对应于复杂任务需要更高的算力调用、更大的推理成本。北美 OpenAI、Anthropic 与高价值场景(金融、法律)的私募基金合作模式,本质上也是 Token 经济在企业级场景的兑现。

从供给端看,英伟达提出的 Token 工厂经济学正在重塑全球数据中心建设方向——未来计算设施将转型为专门生产 Token 的工厂,核心商业竞争指标将从单纯的算力规模让位于每瓦的 Token 吞吐量与能效比。

MiniMax 提出的大模型公司价值公式 "智能密度 × Token 吞吐量" 也呼应了同样的产业方向:智能密度反映模型输出 GDP 的能力,Token 吞吐量则不仅取决于算力规模,还与模型效率、底层架构强相关。

3.4 案例分析:Token 工厂的落地节奏

根据机构调研披露,某头部算力运营商已明确转型 Token 工厂方向,具体规划如下:

公司未来最核心的发展方向不是简单的 AIDC 和算力租赁,而是 Token 工厂模式;

将各类以闭源为主的模型部署在自身算力中心,与模型公司基于 API 调用的 Token 进行分成,公司预计获得 Token 收入的 70%;

今年将在五象云谷落地 4 个万卡集群,二期 AIDC 电力容量将达到一期的 10 倍;

通过多种方式具备 300 亿左右的资金规模来支持算力业务发展。

我们判断,Token 分成、Token 工厂的落地将在 2026 年二三季度密集出现,从单一龙头扩展至云厂商和部分算力服务厂商的整体转型。

四、北美算力共振与中美算力周期的对照

4.1 北美算力涨价逻辑实锤

2026 年五一假期期间,美股算力市场出现强力涨价信号:5 月 4 日,北美 H100 和 B200 算力租赁服务价格分别较 4 月 30 日上涨 22% 和 6%,带动美股数据中心和算力租赁板块大幅走高。这背后是几大云厂明确上调 AI 资本开支指引——谷歌 CEO 在一季报电话会议上明确表示"算力受限、谷歌云收入本可以更高",并预计在未来 24 个月内消化 50% 的积压订单,需求能见度至少延伸到 2028 年。

黄牛市场的报价同样验证供需失衡:4 月中旬一台 B300 服务器现货报价 545 万元 / 4 月 30 日上涨至 643 万元 / 五一假期末突破 700 万元的标题报道。

4.2 中美算力共振的核心逻辑

从映射关系看,北美算力链通过三条主线传导至 A 股:

存储链:涨价 + 满产满销,DRAM/NAND 一年内多次涨价创新高,产业内"从头发丝到后脚跟全都缺"。

CPU 链:服务器 CPU 受 AI 推理拉动结构性供给紧张,涨价幅度可观。

算力租赁链:需求侧确定性最强,且叠加管制属性变相收紧供给,景气度高于存储和 CPU。

4.3 头部玩家的预期差:出厂价稳定、净利率高企

一个被市场忽视的关键预期差是:虽然服务器价格越来越高、算力租赁新签价格水涨船高,但头部算力租赁玩家的拿货价(出厂价)其实没有变。原因在于头部玩家本身就是英伟达全球云服务合作伙伴,可以拿到最便宜的二手货源,而不需要去抢三手 700 万一台的 B300 服务器。

这种"无中间商赚差价"的优势,使得头部算力租赁玩家的净利率比二线厂商高很多,在涨价周期中具备更显著的盈利弹性。

五、风险因素与产业演进展望

5.1 短期风险提示

当前市场对算力租赁板块的关注度极高,但需要警惕几类潜在风险:

乐观预期已较充分:部分公司涨幅已包含较多未来乐观预期,服务器 700 万一台已是大厂测算的盈亏平衡线。

控价、举报、合规风险:大厂可能联手控价、限价采购,或竞争对手相互举报导致非合规渠道收紧。

供给侧改善:H200 可能逐步放开,叠加八九月份开始国产算力卡先进制程产能从翻一倍到翻两倍。

线性外推风险:算力的稀缺性会随着先进制程扩产、模型压缩、推理优化等技术演进被产业链一点点优化掉。

5.2 中长期产业演进:从硬件高溢价到软件平台生态

从 23 年到 26 年的算力超级周期,本质上是"硬件赚钱、软件生态被低估"的阶段。但摩尔定律的经典告诉我们,凡是足够贵、足够重复、足够刚需的东西,最终都会被产业链一点点优化掉。具体路径包括:

模型压缩、量化、蒸馏、稀疏计算;

新架构、编译器优化、推理加速、自研 ASIC、边缘部署;

算法效率提升、云厂商自研芯片、推理成本下降;

开源生态成熟、客户议价能力上升;

一个标志性事件是:大基金(以前主投中芯、长存等硬件)正在领投 DeepSeek(估值逼近 450 亿美元)等软件公司,目的是为了打通"中国设备 + 中国芯 + 中国模型"的产业生态闭环。

这意味着投资视角需要从"硬件稀缺溢价"逐步转向"软件平台生态"——真正长期有复利的公司不能只靠卖昂贵的芯片,还要变成平台、生态和标准。

5.3 投资框架建议:三阶段、三主线

5.4 选股两大原则

原则一:全球前三名。国产算力链中,要么是行业绝对头部,要么是头部公司核心供应商,享受龙头集中度提升的红利。

原则二:国产唯一替代。在卡脖子环节具备国产替代不可替代性的稀缺资产,享受产业链国产化的长期 alpha。

六、总结与展望

国产算力产业链正在迎来从硬件突破、生态闭环到商业模式跃迁的三重共振。短期看,昇腾 950 超节点放量带动半导体国产替代全链条放量;中期看,Token 工厂模式将重塑算力租赁的盈利能力上限;长期看,中国必须通过算力 + 模型 + 应用的整体生态闭环参与全球科技竞争。

面对这一历史性产业窗口,投资者既要把握短期的硬件超额收益机会,更要前瞻布局软件、平台、生态的长周期价值。Token 经济学的兴起,正是这一产业范式转变的标志。

本报告基于公开会议纪要与产业调研整理,仅供内部投研参考,不构成投资建议。
